# 🧩 数独大师 Sudoku Master

一款功能全面的数独学习与练习 Windows 桌面应用。

## ✨ 功能

### 🎮 游戏模式
- **5个难度等级**: 简单、中等、困难、专家、大师
- **智能棋盘**: 点击选择、键盘输入、高亮显示
- **笔记模式**: 在单元格中标记候选数
- **自动候选数**: 一键显示所有可能数字
- **撤销/重做**: 完整的历史记录
- **错误检测**: 实时高亮错误填写
- **提示系统**: 逐步提示，附带技巧解释
- **计时器**: 记录解题时间
- **完成动画**: 解完题目有庆祝效果

### 📚 学习模式
- **12种技巧详解**: 从入门到高级，循序渐进
- **交互式示例**: 每种技巧配有实际案例
- **技巧分级**: 易/中/难 标注

包含技巧：
1. 唯余数 (Naked Single)
2. 隐式唯一 (Hidden Single)
3. 显式数对 (Naked Pair)
4. 隐式数对 (Hidden Pair)
5. 指向数对 (Pointing Pair)
6. 区块删减法 (Box/Line Reduction)
7. 显式三数组 (Naked Triple)
8. 隐式三数组 (Hidden Triple)
9. X翼 (X-Wing)
10. 剑鱼 (Swordfish)
11. XY翼 (XY-Wing)
12. XYZ翼 (XYZ-Wing)

### 🤖 解题器
- **逐步求解**: 观察AI如何一步步用逻辑技巧解题
- **自动求解**: 自动演示完整解题过程
- **手动输入**: 自己输入题目让AI求解
- **步骤详情**: 每一步都有详细的技巧说明

### 📊 统计
- 总游戏数、完成率
- 最佳/平均时间
- 各难度详细统计
- 提示使用统计

### 🎨 其他
- **深色/浅色主题** 切换
- **键盘快捷键**: 数字键、方向键、Shift+数字做笔记
- **本地存储**: 自动保存进度
- **响应式设计**: 适配不同屏幕尺寸

## 🚀 使用方法

1. 克隆本项目
2. 直接用浏览器打开 `index.html`
3. 或者托管到任意静态服务器上

无需安装任何依赖，纯 HTML/CSS/JS 实现。

## ⌨ 快捷键

| 按键 | 功能 |
|------|------|
| 1-9 | 填入数字 |
| 0 / Delete | 擦除数字 |
| 方向键 | 移动选中单元格 |
| Shift + 数字 | 笔记标记候选数 |
| Ctrl + Z | 撤销 |
| Space | 切换笔记模式 |

## 🛠 技术栈

- 纯 HTML5 / CSS3 / JavaScript (ES6+)
- 无框架，无依赖
- LocalStorage 实现数据持久化
- CSS Variables 实现主题切换
